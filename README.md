# LogCheck

LogCheck is a simple Bash script that checks a log file for errors.

The script verifies that a log file exists and then searches for the word "ERROR".

# Purpose

In Linux systems, log files are essential for troubleshooting and monitoring.
LogCheck provides a quick way to verify if a log contains errors before taking further action.

# Usage

./log-check.sh log_file

# Requirements

Bash shell  
Linux system
